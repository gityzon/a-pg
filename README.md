# a-pg

Alist+postgreSQL run on replit.

#### APP:

[Alist-v2](https://github.com/alist-org/alist/tree/v2)

### Packages:

1. bashInteractive
2. pkgs.less (optional)
3. pkgs.postgresql_14 (optional)

### Feature:

Local PostgreSQL_v14 as high-performance database.
